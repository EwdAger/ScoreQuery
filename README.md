# 湖科大教务系统成绩查询（学生版)

班主任版查成绩的试验性作品, 用于学生查询自己的成绩情况. 

### [爬虫部分](https://github.com/EwdAger/ScoreQuery/blob/master/cjcx/jwc_spider.py)

- 模拟登陆至教务网（含[验证码识别](https://github.com/EwdAger/ScoreQuery/blob/master/cjcx/captcha_verify.py)）
- 通过教务网神秘接口获取个人成绩总表
